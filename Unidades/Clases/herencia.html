Introducción a Herencia en la UNC:


En el desarrollo de software, el concepto de herencia es esencial para estructurar y organizar el código de manera eficiente. En la Universidad Nacional de Cajamarca (UNC), donde la gestión de información académica y la adaptabilidad de los sistemas informáticos son cruciales, la herencia se convierte en un componente clave y mejora la enseñanza en la escuela de ING. SISTEMAS.

La herencia es un principio de la programación orientada a objetos que permite la creación de nuevas clases basadas en clases existentes, heredando sus atributos y métodos. En la UNC, esta capacidad de construir sobre estructuras ya establecidas se traduce en la creación de sistemas informáticos más robustos y adaptados a las complejidades de la gestión académica y administrativa.

La herencia facilita la creación de jerarquías de clases que reflejan la estructura de la información en la UNC. Por ejemplo, en sistemas académicos, la herencia puede ser empleada para representar distintos tipos de usuarios, como estudiantes, profesores y personal administrativo, compartiendo características comunes y permitiendo la extensión de funcionalidades específicas.

En la UNC, la herencia se utiliza estratégicamente para promover la reutilización del código, mejorar la claridad y mantenibilidad, y adaptar los sistemas informáticos a las cambiantes necesidades académicas y administrativas, contribuyendo así a la eficiencia y flexibilidad de las aplicaciones desarrolladas.

package herencia;
public class mascotas {
public String raza;
public String nombre;
public String color;
//Constructor: Este constructor se genera después de generar el constructor
//en la clase hijo



public void correr () {

System.out.println("La mascota esta corriendo");
}

public mascotas() {
super();
// TODO Auto-generated constructor stub
}
public void dormir () {
System.out.println("La mascota esta durmiendo");

}

public void comer () {
System.out.println("La mascota esta comiendo");
}
}

Clase Hija 1:


package herencia;

System.out.println("Color de pelo: "+color);

}

public perro(String raza, String nombre, String color, String vacunas) {
